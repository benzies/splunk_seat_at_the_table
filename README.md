## Instructions
1. mkdir -p /opt/splunk/etc/apps/benzies_app_ssatt/
1. cd /opt/splunk/etc/apps/benzies_app_ssatt/
1. git clone 
1. install [Lookup Editor](https://splunkbase.splunk.com/app/1724/)
